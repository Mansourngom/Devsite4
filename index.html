Workflow Git

Comment nous avons collaboré sur GitHub

Ce projet a été réalisé en utilisant Git et GitHub pour gérer les contributions de chaque membre de l'équipe de façon professionnelle.

1
Création des branchesChaque développeur a créé sa propre branche : recette/mansour, recette/khadija...
2
Développement individuelChacun a travaillé sur sa landing page et ses modifications depuis sa branche.
3
Pull RequestsChaque membre a soumis une Pull Request pour merger son travail vers la branche recette.
4
Merge & DéploiementLe chef de projet a validé et mergé toutes les contributions dans la branche principale.
